# Support

RunQ is currently a local alpha open source project.

## Best Places To Ask

- Usage questions: open a GitHub Discussion if enabled, otherwise open a question issue.
- Bug reports: use the bug report issue template.
- Feature requests: use the feature request issue template.
- Security issues: follow `SECURITY.md` and do not open a public issue.

## Before Asking

Please include:

- RunQ version or commit SHA.
- Node.js version from `node --version`.
- Operating system.
- Command you ran.
- Database path, if relevant.
- Redacted output from `node src/cli.js doctor --json --db <path>`.
- Whether the issue affects Claude Code, Codex, OpenClaw, Hermes, or the generic collector.

Do not paste raw prompts, API keys, secrets, private command output, or proprietary code.
